## Special instructions
Please note that these instructions only works in `this interpeter`.\
To use them, put `#` at the top of the file. **It has to be the first character**.

### Special instructions list
`!`: Print the current memory cell value (as an integer)\
`?`: Pause the program: press enter to continue.
